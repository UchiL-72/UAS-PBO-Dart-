## Tema Aplikasi
Sistem Perpustakaan (CLI/Terminal Based)

## Fitur Program
1. **--- Tambah Data ---** Menambahkan data Buku atau Majalah ke sistem.
2. **--- Lihat Semua Data ---** Menampilkan daftar seluruh buku dan majalah yang tersimpan menggunakan konsep Polymorphism.
3. **--- Cari Data ---** Mencari data item berdasarkan kata kunci nama memanfaatkan Higher Order Function (`.where`).
4. **--- Hitung Total ---** Menjumlahkan total harga seluruh item menggunakan Higher Order Function (`.fold`).
5. **--- Simpan Data ---** Simulasi proses penyimpanan data secara asynchronous (`Future`, `async/await`).
6. **--- Validasi Data ---** Proteksi kesalahan input harga negatif atau nama kosong menggunakan Custom Exception.
